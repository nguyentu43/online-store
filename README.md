# Eshop

How to run

- Rename .env.example to .env and add config to it
- Run artisan commands
```bash
php artisan key:generate
php artisan migrate
php artisan db:seed
php artisan jwt:secret
php artisan storage:link
```
- Admin account: eshop@eshop.test - 123456
